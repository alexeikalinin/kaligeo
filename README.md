# KaliGEO — AI Search Visibility Audit

Лендинг-прототип сервиса KaliGEO. Аудит видимости бизнеса в AI-поиске (ChatGPT, Perplexity, Gemini, Google AI Overviews, Claude).

## Локально

Откройте `index.html` в браузере — файл самодостаточный (внешние только шрифты Google Fonts).

## GitHub Pages

В Settings → Pages → Source: `main / root` — сайт будет доступен по адресу
`https://alexeikalinin.github.io/kaligeo/`
