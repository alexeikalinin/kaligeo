# Теги для отката (Rollback Tags)

Откатиться к тегу:
```
git checkout <tag>          # посмотреть
git checkout main           # вернуться обратно
git reset --hard <tag>      # жёсткий откат ветки main (необратимо!)
```

## Как добавить новый тег перед большими правками

```bash
git tag название-тега
```

Пример:
```bash
git tag pre-redesign
git tag before-new-section
git tag v1.0-stable
```

Тег автоматически указывает на текущий последний коммит.
После создания — добавь запись в эту таблицу.
