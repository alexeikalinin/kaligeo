  Варианты: "13/30 мест до пятницы" + обновление с бэкенда, либо убрать и заменить на реальный дедлайн.

- [ ] **Переставить секции: Problem после Solution** `#ux-flow`

  Новый порядок: Hero → Logos → Solution → Problem (цифры как обоснование) → Deliverables → How it Works → Proof → Pricing → CTA → FAQ

  Решение: на мобилях — inline-подпись под mockup'ом вместо позиционированного tooltip'а.

- [ ] **Аватары в отзывах** `#trust`
  Буквы в сером круге выглядят как плейсхолдер. Минимум: градиент по brand-цвету.
  Максимум: реальные фото с разрешения клиентов.

- [ ] **Hover + active на CTA-кнопках** `#micro-interactions`
  Добавить `scale(1.02)` на hover и `scale(0.98)` на :active для `.btn-accent` и `.btn-primary`.

- [ ] **Активное состояние в навигации** `#ux`

## 🔵 Технический долг

- [ ] **Dark mode поддержка** `#tech`
  CSS-переменные уже готовы к dark mode (есть ветка в Tweaks panel).
  Добавить `@media(prefers-color-scheme:dark)` и переключатель в nav.

- [ ] **Оптимизация шрифтов** `#performance`
  Три Google Fonts (Instrument Serif + Inter + JetBrains Mono) — три отдельных HTTP-запроса.
  Рассмотреть self-hosting или subset через `font-display: swap`.

- [ ] **Meta / OG теги** `#seo`
  Нет `<meta name="description">`, `og:image`, `og:title`.
  Критично для шаринга в соцсетях и мессенджерах.

- [ ] **Sitemap + robots.txt** `#seo`
  Отсутствуют. Нужны для индексации, особенно с учётом того, что сервис помогает с AI-видимостью.

- [ ] **Analytics подключение** `#tracking`
  Нет Яндекс.Метрики, Google Analytics, PostHog.
  Без аналитики невозможно проверить реальную конверсию и A/B тестировать улучшения.

- [ ] **Form backend** `#tech`
  CTA-форма сейчас делает `event.preventDefault()` — данные никуда не отправляются.
  Нужен Formspree / Tally / Supabase endpoint.
